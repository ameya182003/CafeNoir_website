# Cafe Noir

Developed by [Ameya Modak](https://www.linkedin.com/in/ameya-modak/).

Welcome to Caf√© Noir, a virtual journey into the heart of our establishment. This website, built using ReactJS for the frontend and MongoDB for the backend, invites you to immerse yourself in the inviting atmosphere of our home page, learn about our journey in the 'About' section, and stay updated on coffee culture through our vibrant blog.

## Features
- User-friendly navigation menu for seamless exploration
- 'About' section to learn about our journey
- Vibrant blog section to stay updated on coffee culture
- Login page for accessing the blog section; new users can register first

## Technologies Used
- Frontend: ReactJS
- Backend: MongoDB
- Deployment: MongoDB Atlas

## How to Run
1. **Frontend:**
   - Open a terminal and navigate to the frontend directory.
   - Run the following command:
     ```
     npm start
     ```

2. **Backend:**
   - Open another terminal window or tab.
   - Navigate to the backend directory.
   - Run the following command:
     ```
     nodemon app.js
     ```
